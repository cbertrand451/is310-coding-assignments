# Escape the Algorithm

## Social Media + Pop Culture Maze

By Colin Bertrand

## Overview
This maze is themed around social media and algorithm-driven internet behavior.

Your goal is to navigate through directories and files
using the command line to escape the algorithm.

## Necessary Commands
- ls
- ls -a
- cd
- cat
- pwd

## Tips
- Read files carefully — clues are everywhere
- Some things are hidden
- If you feel trapped, that’s intentional

Good luck.
